# tDiary contrib パッケージ

このパッケージには、tDiaryに関係したいくつかのユーティリティやプラグ
インが収録されています。ドキュメントはそれぞれのツールのディレクトリ
に含まれています。

tDiaryで使う場合は、tDiaryのトップディレクトリに Gemfile.local という
ファイルを作成し、以下のように記述します:

```
gem 'tdiary-contrib'
```
